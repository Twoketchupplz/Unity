## Text (Script)
Text의 크기는 Rect Transform 컴포넌트의 Width와 Height로 설정한다. 문자의 크기가 더 크면 문자가 잘리게 된다.
프로퍼티 | |
--|--
__Text__ | 표시할 문자를 입력받는다.
__Character__ | 폰트의 종류, 스타일, 크기, 색을 정할 수 있다.
__Paragraph__ | 정렬방식을 지정한다.
