# 카드 컨트롤 UI
## UI 설명 요약
- `덱`을 누르면 `핸드`에 카드가 채워진다.
- `카드`를 드래그 후 `핸드` 영역 내에 드랍하면 원래 자리로 돌아간다.
- `카드`를 드래그 후 `핸드` 영역 밖에 드랍하면 `카드`가 사라진다.
- `핸드` 내 `카드`는 언제나 중앙 정렬된다.
